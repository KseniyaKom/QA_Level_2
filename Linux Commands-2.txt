Termial. HW_2
1. mkdir dir_1 - Сделать папку dir_1
2. cd dir_1 - Зайти в папку dir_1
3. mkdir inner_dir_1 - Создать папку inner_dir_1
4. pwd - Посмотреть где ты находишься
5. touch tf_1.txt - Находясь в папке dir_1 создать пустой текстовый файл tf_1.txt
6. cat > tf_2.txt (ctrl+c) - Находясь в папке dir_1 через команду cat создать текстовый файл tf_2.txt со следующими строками:
- the first 1
- the second 2
- the third 3
7. cd inner_dir_1 - Зайти в папку inner_dir_1
8. cat > tf_3.txt (ctrl+c) - Через cat сделать текстовый файл tf_3.txt  c любыми строками
9. cat >> tf_3.txt (ctrl+c) - Через cat добавить в текстовый файл tf_3.txt строку “the second 2”
10. cat >> tf_3.txt (ctrl+c) - Через cat добавить в текстовый файл tf_3.txt строку “the sec 2”
11. cat >> tf_2.txt (ctrl+c) - Через cat добавить в текстовый файл tf_2.txt строку “the sec 3”
12. cat >> tf_3.txt (ctrl+c) - Через cat добавить в текстовый файл tf_3.txt строку “the SeCoNd 2”
13. cat >> tf_2.txt (ctrl+c) - Через cat добавить в текстовый файл tf_2.txt строку “the seConD 2”
14. cat > tf_4.txt (ctrl+c) - Сделать текстовый файл tf_4.txt в котором будет 15 строк.
15. cat > tf_5.txt (ctrl+c) - Сделать текстовый файл tF_5.txt в котором будет 13 строк.
16. ls -la - Вывести список всех файлов в папке.
17. cd .. - Выйти из папки inner_dir_1
18. cat inner_dir_1/tf_3.txt - Вывести содержимое файла tf_3.txt в терминал.
19. find . -name "tf_4.txt" - Найти путь к файлу tf_4.txt
20. cat > tf_4.txt (ctrl+c) - Отчистить файл tf_4.txt от содержимого без удаления самого файла.
21. find . -name "*tf*" - Найти путь к файлам у которых есть  “tf” в названии.
22. find . -iname "*tf*" - Найти путь к файлам у которых есть  “tf” в названии и буквы в любом регистре.
23. grep "sec" * - Найти строки в файлах где есть комбинация букв “sec” в текущей папке
24. grep -i "sec" * - Найти строки в файлах где есть комбинация букв “sec” в любом регистре в текущей папке
25. grep -w "sec" * - Найти строки в файлах где есть только комбинация букв “sec” в текущей папке
26. grep -iw "sec" * - Найти строки в файлах где есть только комбинация букв “sec” в любом регистре в текущей папке
27. grep -r "second" * (-r позволяет искать и во внутр папках) - Найти строки в файлах где есть комбинация букв “second” в текущей папке
28. grep -i "second" * - Найти строки в файлах где есть комбинация букв “second” в любом регистре в текущей папке
29. grep "second" ./*/* - Найти строки в файлах где есть комбинация букв “second” во всех папках ниже уровнем
30. grep -rl "second" . - Найти только путь и название файла в строках которых есть комбинация букв “second” в текущей папке
31. grep -rv "second" . - Найти все строки во всех файлах где нет комбинации “second”
32. grep -rlv "second" . - Найти только название и путь к файлам где нет комбинации “second”
33. tail -n4 tf_3.txt - Вывести в терминал 4 последних строк любого текстового файла
34. head -n4 tf_3.txt - Вывести в терминал 4 первые строки любого текстового файла.

39. grep -rl "sec" ./ | xargs -I{} rm -f {} - Команда в одну строку. Удалить текстовые файлы у которых в содержимом есть слово “sec”
